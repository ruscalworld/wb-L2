Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]
```

Срез `[1:4]` будет содержать элементы массива, имеющие индекс 1 <= i < 4. В нашем случае это 77, 78 и 79.
